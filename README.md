# SnakeGame

A simple Snake Game developed in Java.

## Features
- Classic snake gameplay
- Score system
- Keyboard controls
- Game over detection
- Simple Java GUI interface

## Technologies Used
- Java
- Java Swing / AWT
- SQLite

## Project Structure
```bash
SnakeGame/
│── GameFrame.java
│── MainMenu.java
│── SnakeGame.java
│── slhx.sqlite
│── README.md
```

## How to Run

### 1. Clone repository
```bash
git clone https://github.com/tramanhhuynh/SnakeGame.git
```

### 2. Open project
Open the project in:
- IntelliJ IDEA
- Eclipse
- VS Code

### 3. Run the game
Run:
```bash
SnakeGame.java
```

## Controls
| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
